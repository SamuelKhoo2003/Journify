
# Journify

A new way to record your daily thoughts, achievements and keep track of it all!

### Journify In Action!

## Getting Started
First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
